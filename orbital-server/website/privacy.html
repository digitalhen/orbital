Orbital — Privacy Policy

Last updated: April 1, 2026

Overview

Orbital is a macOS menu bar app that displays publicly available NASA mission data. We respect your privacy and collect minimal data.

Data Collection

Orbital does not collect, store, or transmit any personal information. The app does not require an account or login.

Network Requests

The app makes network requests to:

  • NASA/Google Cloud Storage — to fetch publicly available spacecraft telemetry data
  • JPL Horizons API — to fetch Moon position data for distance calculations
  • Cleartext Labs API — to fetch mission configuration (mission name, phases, data source URLs)

No personal data is included in these requests.

Analytics

The app may use Google Analytics (GA4 Measurement Protocol) to collect anonymous usage data such as app launches and feature usage. No personally identifiable information is collected. You can disable analytics by blocking network access to google-analytics.com.

Local Storage

The app stores your preferences (unit selection, enabled metrics, launch-at-login) locally on your device using macOS UserDefaults. This data never leaves your device.

Third-Party Services

The app displays data from NASA's public telemetry feeds. Orbital is not affiliated with or endorsed by NASA.
